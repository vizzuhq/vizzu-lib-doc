    Interface: AxisLabel

    vizzu.Styles.AxisLabel

    Style settings for the values shown on the axis to display the scale being used or the categories along the axis. Note: textAlign has no effect on the Axis label.

    Hierarchy

    • OrientedLabel

      ↳ AxisLabel
